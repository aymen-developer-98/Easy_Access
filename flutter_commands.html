This file contains the most used commands in flutter framework

To use two commands togather use && syntax like the example below

flutter pub get && flutter run

Use flutter firebase from flutter cli without get json file and set it in the project

the the documantation in this link

Commands to build your project

  • flutter build apk --split-per-abi
  • flutter build appbundle
  • flutter build appbundle --no-shrink

Commands to check code health

  • flutter test
  • flutter analyze

Commands to run the project

  • flutter run --release
  • flutter run

Commands to get log from runing app

  • flutter logs

Commands to manage packages of your project

  • flutter clean
  • flutter pub get
  • flutter pub cache repair

Commands to run the code generator

  • flutter pub run build_runner build --delete-conflicting-outputs
  • flutter pub run build_runner watch --delete-conflicting-outputs

The usege of some packages

icons package

flutter_launcher_icons: ^0.9.2
then set this code in yaml

flutter_icons: 
  android: "launcher_icon" 
  ios: true 
  image_path: "assets/appicon.png"

then run

  • flutter pub run flutter_launcher_icons:main -f pubspec.yaml

name package

rename: ^2.1.1
and use commands

  • flutter pub global activate rename
  • flutter pub global run rename --bundleId dz.makay.mitaa
  • flutter pub global run rename --appname "mitaa"
  • flutter pub global run rename --appname YourAppName --target ios
  • flutter pub global run rename --appname YourAppName --target android
  • flutter pub global run rename --appname YourAppName --target web
  • flutter pub global run rename --appname YourAppName --target macOS
  • flutter pub global run rename --appname YourAppName --target windows
