Commands related to branch

Switch to another local branch

git switch myLocalBranch

Used to get the list of local branches

git branch

Switch to another branch or create it localy if does not exist

git checkout −b myLocalBranch

Push local branch to remote

git push −u origin myLocalBranch

-u, --set-upstream-to

Delete local branch

git branch −d myLocalBranch
