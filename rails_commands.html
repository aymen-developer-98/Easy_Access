This file contains the most used commands in rails framework

to create rails api

rails new hospital_appointment_ticket_api --api

to create rails app

rails new your_app_name

to create the migration of active storage

rails active_storage:install

to create migration file

rails generate migration CreateFirstSchema

when use db use this command

  • rails db:drop db:create db:migrate
  • rails db:setup

when use db use this command

rake db:migrate

to generate route file use this comand

rails routes

to run the api use

  • rails s -b 0.0.0.0
  • rails s -e production -b 0.0.0.0
  • rails server
